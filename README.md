## Pre Required
- npm/yarn
- expo-cli

## Installation

1. Get a free API Key at [https://openweathermap.org/api](https://openweathermap.org/api)
2. Clone the repo
```sh
git clone https://github.com/BarbierAlexis/Little-Swiss-Knife.git
```
3. Install NPM packages
```sh
npm install
```

4. Enter your API in `config_default.js`
```JS
const apiKeyWeather = 'apikeyhere';
export {apiKeyWeather};
```

## Start

- For launch the application use
```sh
npm start
``` 
or 
```sh
expo start
```
